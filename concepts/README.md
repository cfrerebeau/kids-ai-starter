## How badges work here

You don't earn a badge by saying "a variable is like a box." That's just words. You earn it by **using a variable in your own project, in a way the program depends on**. Claude checks. The artifact is the proof.

This is on purpose. Words are easy. Code is real.

## How to read these

Each concept file is short — under a page. Read it, then go back to building. If you have a "why" question after reading, ask Claude. That counts toward the *AI Interrogator* badge.
