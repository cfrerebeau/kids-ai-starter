# Tinker

Working examples. Run them. Break them. See what happens.

## Files

- **`hello.html`** — Open in your browser. Pixels in 5 seconds. No install.
- **`api_demo.py`** — Run in the terminal. Talks to the real internet, shows you something different each time. Needs Python.
- **`hello.py`** — The classic. Three lines. Run in the terminal.
- **`streamlit_hello.py`** — A tiny web app, in Python. Needs `streamlit` installed (ask your coach).

## How to run them

### `hello.html`

Double-click the file. Or right-click → "Open with" → your browser. Done.

### `api_demo.py` and `hello.py`

```
python tinker/api_demo.py
```

or

```
python tinker/hello.py
```

If you get "command not found: python", try `python3` instead.

### `streamlit_hello.py`

You need streamlit installed first. That's a grown-up step — ask Claude, it'll know what to say to your coach.

Once installed:

```
streamlit run tinker/streamlit_hello.py
```

A page will open in your browser.

## What now?

Pick one. Change something in it. See what happens. The point isn't to understand every line — the point is that **you made the computer do something different**. That's what code is.

Tell Claude what you tried and what changed.
