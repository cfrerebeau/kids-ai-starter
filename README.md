# Kids AI Starter

Learn to ship a real AI app, with Claude as your coach.

For kids ~12–16 and a parent who can sponsor the journey. The repo guides you through three skill sets at once: **entrepreneurship** (finding a real problem, talking to real users), **building** (Python, the web, databases), and **AI fluency** (using Claude well, verifying its output, building your own context).

## Who this is for

- A kid who wants to make something real.
- A parent willing to be the weekly coach (~30 min Sundays) and handle a few install/account moments.

## Quickstart

1. **Parent:** read `PARENTS.md` first — Section 0 walks through the one-time setup.
2. **Parent:** click **"Use this template"** on this repo's GitHub page → create the kid's private copy.
3. **Parent:** install VS Code + Claude Code on the kid's machine (steps in `PARENTS.md` Section 0).
4. **Kid:** clone their new repo, open in VS Code, open `START_HERE.md`.
5. **Kid:** in the VS Code terminal, type `claude`. Claude takes it from there.

## What you'll build

Whatever you want. A real app, on the public internet, that at least one stranger uses. The repo doesn't tell you what to build — it helps you find out what's worth building.
