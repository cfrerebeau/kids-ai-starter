# Quests

The full game. Browse it whenever you want.

## How it works

- Every quest is worth **XP**. Finish a quest → earn XP → level up eventually.
- Every concept comes with a **badge**. Earn the badge by *making something with the concept*, not by explaining it.
- Some badges are **behavior badges** — for things like pushing back on Claude when you're right, or catching Claude making a mistake. Those need your coach to cosign.
- You don't pick a quest. Claude does, based on where you are. Your current quest is in `current.md`. Your history is in `completed.md`.

## Concept badges (6)

Earn by making something in your own project files.

| Badge | How to earn |
|---|---|
| *I Know What Code Does* | Write a Python file from scratch that runs (not modifying an existing template) |
| *Variable Whisperer* | Use a variable you named yourself, in a way the program depends on |
| *Function Caller* | Write a function and call it. Bonus: a function calling another function |
| *Where Does Code Run?* | Build one thing in a browser AND one thing in a script. Show both. |
| *The Memory Place* | Build something that persists across runs (a file, a DB — anything) |
| *AI Verifier* | Find and record one Claude output that was wrong (coach cosigns) |

## Behavior badges

Awarded for *how you work*, not what you make.

| Badge | When |
|---|---|
| *Skeptic* | First time you pushed back on Claude without being prompted |
| *AI Interrogator* | Asked 5+ "why" questions in a single session |
| *Bug Catcher* | Caught Claude in a real mistake (coach cosigns) |
| *Receiver of No* | First user who told you "I wouldn't use this" |
| *Listener* | Kept a feature request log from users |
| *Restarter* | Abandoned a bad idea on purpose |
| *Held Position* | Pushed back on Claude and you turned out right (coach cosigns) |
| *Evolver* | Changed your persona because real users disagreed |
| *Shipper* | First stranger uses your app |

## Where everything lives

- `current.md` — your active quest right now.
- `completed.md` — everything you've finished. Your coach reads this on Sundays.
